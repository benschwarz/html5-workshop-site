For theloveofthe web

An adaptation for the modern web

A practical HTML5/CSS Workshop

At this workshop you’ll learn about new elements, browser techniques, working with styles using new and emerging web standards, and an introduction to modern web typography.

If you’ve ever wondered if you’re really doing it right, have become cynical to the ways of the 'semantic' or 'open web' or just plain want to get into sync with the platform you work with every day, then this workshop is for you.

Assumed knowledge is that you’ve written HTML before, the focus will be on the future, not on the past, but everything will be practical and usable now.
